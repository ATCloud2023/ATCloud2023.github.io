ATCloud
Estaçao Meteorológica

Objetivo

Criação - visando a sustentabilidade - de uma estação meteorológica dedicada à área da prática de esportes radicais, onde é imprescindível ter em mãos os dados do clima/tempo para garantir a segurança, e otimizar a experiência esportiva.


O Prototipo

O ESP32 é uma escolha excepcional para uma ampla gama de projetos devido à sua versatilidade e recursos avançados. Este microcontrolador oferece uma combinação poderosa de desempenho, conectividade e eficiência energética, tornando-o ideal para uma variedade de aplicações.

Principais Componentes

  • Microcontrolador ESP32
  • Sensor de Temperatura e Umidade (DHT11)
  • Sensor de Pressão (BPM280)
  • Anemômetro (Proprietário)
